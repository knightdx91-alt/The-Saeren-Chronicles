## Purpose

This file replaces the three-way stuck-point split (cognitive / motivational / physiological) with an expanded 8-type taxonomy. It defines the diagnosis protocol, perfectionism modifier, intervention selection logic, and nocebo inoculation requirement.

The coaching protocol references this file. The writer never sees it directly.

---

## Primary Block Types

### Type 1: Cognitive Overload

**What is stuck:** The writer cannot hold the moving parts or causal dependencies in mind.

**Signals:**
- "Every time I fix one thing, another breaks."
- "I know what's wrong but I can't figure out how."
- Multiple interdependent variables exceeding working memory.

**Prompt suitability:** High.

**Intervention:** Structural prompts — Constraint, Isolation, Scale-shift. Decompose the problem into smaller independent pieces. Narrow to a single corridor. Freeze everything except the one variable under test.

---

### Type 2: Motivational / Avoidance

**What is stuck:** The writer is avoiding exposure, failure, or effort — not failing at the work itself.

**Signals:**
- "I hate this scene."
- "I keep starting and stopping."
- "I don't know if I can pull this off."
- Procrastination patterns, emotional language about the draft.

**Prompt suitability:** Conditional.

**Intervention:** Existing exercise library first (Name the Resistance, Permission Draft, Sit with the Draft, Write Toward the Difficulty, Inventory What Works, Seek Outside Readers). Structural prompts only after motivational intervention fails or if the avoidance is specifically about a structural decision (e.g., the writer is avoiding a scene because they don't know how to restructure it, not because they fear it emotionally).

---

### Type 3: Physiological / Environmental

**What is stuck:** The writer is too depleted to benefit from more cognitive work.

**Signals:**
- "I can't focus."
- "I've been at this for weeks."
- "Life stuff is getting in the way."
- Declining quality without declining effort.

**Prompt suitability:** Low. Usually no prompt.

**Intervention:** Coach away from the desk. Recommend a real break (days, not hours). Adjust session scope to match diminished capacity. If a deadline exists, use deadline coaching to draw an honest achievability line. Do not motivational-coach a burned-out writer. Do not generate structural prompts — more structure becomes more pressure.

---

### Type 4: Decisional / Equipoise

**What is stuck:** The writer sees multiple viable options and cannot choose. Different from cognitive overload — the problem isn't too many variables, it's genuine equipoise between alternatives.

**Signals:**
- "I could go either way."
- "Both versions work."
- "I've been going back and forth for days."
- Three valid restructuring options with no clear winner.

**Prompt suitability:** High.

**Intervention:** Structural prompts — Scale-shift, Temporal, Constraint. The intervention is satisficing heuristics: prototype briefly, commit to the one with more energy, treat the decision as reversible. Scale-shift is especially useful: compress competing options to see which retains more structural energy.

---

### Type 5: Identity / Vision Threat

**What is stuck:** The revision threatens the writer's self-concept or the book they set out to write.

**Signals:**
- "If I cut this, it's not the book I wanted."
- "This revision betrays my vision."
- "I can't write a book that does that."
- Defensive framing that sounds like aesthetic disagreement but is actually grief or self-protection.

**Prompt suitability:** Conditional.

**Intervention:** Reframe first — always. Externalize the manuscript from the self. Name the grief for the version being revised away. Acknowledge that the writer's attachment may be structurally correct (the part they're protecting may actually be load-bearing).

Then, if reframing is insufficient, offer low-stakes Constraint or Perspective prompts framed explicitly as temporary experiments the writer can reject. The clinamen clause is especially critical here — the writer must feel genuine permission to refuse the experiment's results.

**Failure mode:** Jumping to a structural prompt without reframing first. This reads as the tool overriding the writer's artistic judgment.

---

### Type 6: Aesthetic / Execution Gap

**What is stuck:** The writer can diagnose the structural fix but cannot execute it at the quality level they want.

**Signals:**
- "I know the scene needs X but I can't make it work."
- "I hear it in my head but it won't come out right."
- Attempts and restarts, not avoidance. The writer is trying, not hiding.

**Prompt suitability:** Conditional.

**Intervention:** Isolation prompts to practice the specific craft skill without global coherence burden. Permission Draft if perfectionism is amplifying. Not cognitive restructuring (they understand the problem). Not motivational coaching (they're not avoiding).

**Note:** Under Rose's (1984) definition, this is technically not writer's block — it's a skill gap. The intervention is closer to craft instruction than cognitive restructuring. The module offers Isolation prompts as practice space, not as therapy.

---

### Type 7: Accumulated-Feedback / Overmediated

**What is stuck:** Too much input has destroyed the writer's compass. Every sentence arrives pre-tagged with problems. The diagnostic overlay has replaced direct encounter with the text.

**Signals:**
- "I can't read my own work anymore."
- "I keep hearing all the notes."
- "I don't know what I think."
- The writer references diagnostics, feedback, or notes more than the manuscript itself.

**Prompt suitability:** Low. **This is the highest-risk block type for APODICTIC.**

**Intervention:** Prompts usually worsen this state. Preferred interventions:
1. Strip back support — fewer layers of mediation, not more.
2. Sit with the Draft (existing exercise) — reread without notes, pen, or keyboard.
3. Inventory What Works (existing exercise) — rebuild positive compass.
4. Seek Outside Readers — a fresh encounter from someone without the diagnostic overlay.
5. Short break from the project.

If anything is offered at all, one very low-load Isolation prompt is the maximum — and only when the writer explicitly asks for something structural to do.

**Iatrogenic risk:** This block type can be caused by the diagnostic tool itself. The module must recognize when APODICTIC's own output is the problem and respond by reducing output, not adding more.

---

### Type 8: Stage-Mode Mismatch

**What is stuck:** The writer is applying the wrong level of attention to the draft's current stage.

**Signals:**
- Polishing prose on a structural draft.
- Structural critique on near-finished work.
- Premature line editing.
- Revision energy spent at the wrong altitude.

**Prompt suitability:** Moderate.

**Intervention:** Often solved by reframing before prompting:
- "This is a structural draft — polish is premature. The prose will change when the structure settles."
- "The structure is stable — you're in polish mode now. Trust the architecture."

If prompting, Scale-shift to recalibrate the unit of work (compress to find structural load, or expand a single line to find the polish opportunity).

## Perfectionism Modifier

Perfectionism is not a block type. It is a severity amplifier that cross-cuts all eight types.

**Detection signals:**
- Excessive self-criticism disproportionate to actual draft quality.
- Repeated restarts of the same passage.
- "It has to be perfect."
- Reluctance to commit to any version.
- Chronic comparison to published work or idealized internal standard.

**When present, adjust the primary intervention:**

1. Add Permission Draft framing ("this is disposable, not canonical") to any structural prompt.
2. Lower the stakes language in the session plan.
3. Emphasize the clinamen clause more heavily — the writer needs extra permission to deviate.
4. Consider whether the perfectionism itself, rather than the structural problem, is what needs addressing first. If the writer can't attempt any version because all versions feel inadequate, the perfectionism is the block, and the motivational exercise library (Name the Resistance, Permission Draft) should precede structural prompting.

## Two-Step Diagnosis Protocol

The expanded taxonomy requires a two-step diagnosis before intervention selection.

### Step 1: Gate Question

Is this cognitive, motivational, physiological, or something else?

**Gate tests:**
- **Cognitive:** The writer can articulate the problem and is trying to solve it, but can't. The work feels too complex, not too threatening.
- **Motivational:** The writer is avoiding, not failing. The work feels threatening, not complex.
- **Physiological:** The writer is depleted. No amount of coaching or prompting will help until the resource deficit is addressed.
- **Something else:** The block doesn't fit the three primary categories. Proceed to refinement.

### Step 2: Refinement Question

*What kind* of cognitive / motivational / other block is this?

| Gate answer | Refinement options |
|-------------|-------------------|
| Cognitive | Overload (Type 1) vs. Decisional (Type 4) vs. Stage-mismatch (Type 8) |
| Motivational | Avoidance (Type 2) vs. Identity threat (Type 5) vs. Aesthetic gap (Type 6) |
| Physiological | Depletion (Type 3) — no refinement needed |
| Something else | Accumulated-feedback (Type 7) — check for iatrogenic risk |

### Step 3: Modifier Check

Is perfectionism amplifying the block? If yes, adjust the intervention per the modifier protocol above.

### Step 4: Intervention Selection

Based on the refined type, select the intervention layer:

| Block type | Primary intervention | Prompt family if prompted |
|-----------|---------------------|--------------------------|
| 1 — Cognitive overload | Structural prompt | Constraint, Isolation, Scale-shift |
| 2 — Motivational/avoidance | Existing exercise library | Prompt only if avoidance is about a structural decision |
| 3 — Physiological | Coach away from desk | **No prompt** |
| 4 — Decisional | Structural prompt | Scale-shift, Temporal, Constraint |
| 5 — Identity/vision | Reframe first, then low-stakes prompt | Constraint, Perspective (clinamen critical) |
| 6 — Aesthetic/execution | Isolation prompt | Isolation, Permission Draft framing |
| 7 — Accumulated-feedback | Strip support, existing exercises | **Usually no prompt** |
| 8 — Stage-mismatch | Reframe first, then Scale-shift | Scale-shift |

### Step 5: No-Prompt Check

Before generating any structural prompt, verify the state is not in a no-prompt zone:

| State | Action |
|-------|--------|
| Physiological depletion / burnout | No prompt. Rest, rescope, deadline honesty. |
| Clear evidence-acquisition gap | No prompt. Evidence queue, verification. |
| Accepted cut decision | No prompt. Cut and propagate. |
| Factual or continuity error | No prompt. Fix directly. |
| Accumulated-feedback saturation | Usually no prompt. Strip back support. |
| Locked Keep/Cut decision (not reopened by writer) | No prompt. Respect the lock. |
| State file missing or empty | No prompt. Route to `/start`. |

## Nocebo Inoculation Requirement

Every block diagnosis or structural problem identification must include an acknowledgment that the writer may already be handling this. This is a hard constraint, not a style suggestion.

**Required language pattern:** When delivering a block diagnosis, include one sentence acknowledging that the writer may already be navigating this. Frame the diagnosis as one possible lens, not the definitive truth about the manuscript or the writer.

**Examples:**
- "This looks like it might be a decisional block — you're seeing multiple valid options and can't commit. But you may already be working through this in a way I can't see from the diagnostic."
- "The diagnostic suggests cognitive overload here — too many moving parts. If that doesn't match what you're experiencing, trust your read over mine."

**Theoretical basis:** Sandra et al. (2025) showed that awareness workshops caused healthy adults to falsely self-diagnose; brief nocebo education halved the effect. Kalyuga et al. (2003) on the expertise reversal effect: instructional techniques effective for novices can harm experienced learners.

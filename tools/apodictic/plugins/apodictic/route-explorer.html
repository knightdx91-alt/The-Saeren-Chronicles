Projects & resume. Runs are registered as projects. Come back later and /start <id> resumes from the project's lifecycle state with zero intake questions; /projects lists, inspects, and tidies what you have registered.

Execution mode. Any route can run single-agent (default) or, for final submission prep, in swarm mode — passes run in architectural isolation as verification insurance, not a higher raw yield. Swarm costs roughly 5x the tokens (measured up to ~8.5x on long fiction); reserve it for the final round and the writer/verifier spot-check.

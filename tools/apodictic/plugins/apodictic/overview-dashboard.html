APODICTIC

Development Editor for fiction and argument-shaped nonfiction — reads what you wrote, diagnoses structure, never rewrites.

New here? This page is a map of everything the editor can do — you don't need to memorize it. Open your agent in your manuscript's folder and type /start; three questions route you to the right workflow. Writing an argument-shaped piece? /start still works — answer "nonfiction" at the constraint question to reach the Nonfiction Argument Engine.
The Firewall: This tool diagnoses structure. It identifies problems, explains mechanisms, and proposes classes of solution. It never invents content — no plot ideas, no dialogue, no prose, no arguments. Creative and argumentative authority stays with the author.

Workflow

Q1 determines what you're working with:
  • Idea — no prose yet (logline, premise, notes) → routes to Pre-Writing Pathway
  • Fragments — disconnected scenes/sketches, no continuous narrative → Fragment Synthesis
  • Partial — draft in progress, not finished → Core DE with partial-manuscript flag
  • Complete draft — beginning, middle, and end exist → Core DE
  • Series — multiple books in a shared world → series-aware routing
Q2 options change based on Q1. For a complete draft:
  • Figure out what's wrong → Core DE (general diagnostic)
  • Check submission readiness → Core DE → Pass 11
  • Clean up AI-assisted prose → Core DE + AI-Prose Calibration
  • Sort through beta reader feedback → Feedback Triage → Core DE
Q3 modifies how the workflow runs:
  • On a deadline → Fast Triage (Pass 1 only, 3 interventions max)
  • Parts written with AI → adds AI-Prose Calibration overlay
  • Nonfiction → nonfiction-specific workflow adjustments
  • Sensitive/risky content → adds the Legal Risk Register: flags defamation, privacy, and rights-clearance exposure and routes serious items to counsel (not legal advice)
  • Editing someone else's work → editor-facing output framing

What It Analyzes — Macro Blocks

The system runs the passes your question requires, grouped into blocks you can understand.

Structure Map
"Is the structure working?"
Pass 0: Reverse Outline Pass 2: Structural Mapping
Beat map, causal chain, proportional analysis. Identifies structural breaks, missing acts, proportion problems.
Reader Dynamics
"Does the pacing hold?"
Pass 1: Reader Experience Pass 3: Rhythm & Modulation
Naive-reader tracking: confusion, boredom, delight, belief breaks. Quantitative pacing analysis.
Character Architecture
"Are my characters landing?"
Pass 5: Character Audit Pass 7: POV & Voice
Psychology engine (Wound→Lie→Want→Need→Fear), agency quotient, arc tracking, voice distinctiveness, POV distribution.
Emotional Dynamics
"Does it land emotionally?"
Pass 4: Emotional Value
Emotional value charge per scene, earned vs. unearned beats, sentiment trajectory, and modulation of intensity across the arc.
Scene Delivery
"Are the scenes doing their jobs?"
Pass 6: Scene Function Pass 7: POV & Voice
Scene legitimacy, function verification (8 scene functions), scene-sequel mechanics.
Reveal Economy
"Is the information flow right?"
Pass 8: Reveal Economy
Information pressure, clue economy, fairness tests, dramatic irony tracking.
Theme & Continuity
"Does it cohere?"
Pass 9: Thematic Coherence Pass 10: Entity Tracking
Motif alignment, controlling idea, continuity of people/places/rules/facts.
Submission Readiness
"Is this ready?"
Pass 11: Critical Quality & Market Viability
Prose quality diagnostic, critical verdict, market reality check, first-50 conversion gate, revision economics.

Common Workflows

I have an argument-shaped piece (op-ed, brief, testimony) and need to know if it holds up
/start → full_draft → repair → constraint: nonfiction → Nonfiction Argument Engine (Dialectical Clarity + Argument companions)
I have a complete draft and want to know what's wrong
/start → full_draft → repair → Core DE (passes 0, 1, 2, 5, 8) → Synthesis
My characters feel flat
/start → full_draft → repair (targeted: characters) → passes 0, 1, 5, 7 → Character Architecture audit
I have an idea and need to figure out the structure
/start → idea → draft → Pre-Writing Pathway
Is my horror novel ready to submit?
/start → full_draft → submit → all passes + Pass 11 + Horror Craft audit + Shelf Positioning
I'm stuck mid-draft and don't know why
/start → partial → repair (diagnostic) → Core DE (partial flag) → Synthesis
I used AI to draft and need to fix the voice
/start → full_draft → repair → constraint: AI → Core DE + AI-Prose Calibration
I have beta reader / critique feedback to sort through
/start → full_draft → feedback → Feedback Triage (sort, validate & prioritize each note) → Core DE with feedback as evidence
My content is sensitive or legally exposed
/start → full_draft → repair → constraint: risk → Core DE + Legal Risk Register (flags exposure, routes to counsel — not legal advice)
I want a beta read that actually returns useful feedback
Beta-Reader Instrument → turns the diagnosis's open uncertainties into a focused, non-leading reader questionnaire
I'm coming back to a project I started earlier
/start <id> → state-driven resume (zero intake questions) from the project registry

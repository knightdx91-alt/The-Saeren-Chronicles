# Editorial Package — The Saeren Chronicles, Book One: The Hazel Years

Submission-ready editorial materials for agents and acquiring editors. Current as of the
round-3 revision pass (2026-06-21). Manuscript: **~86,000 words** (86,726 exact), 18 chapters,
YA epic/school fantasy, Book One of a planned trilogy, standalone-with-series-potential.

## Contents

| File | What it is | Use it for |
|------|------------|-----------|
| `logline.md` | One-sentence pitch + 3 alternates (stakes / irony / theme) | Pitch events, the top of a query, comp-title conversations |
| `query-letter.md` | Full query: hook, ~200-word summary, comps, metadata. Bio + personalization are bracketed placeholders | Agent submissions |
| `synopsis.md` | 1-page and 3-page synopses (full spoilers, incl. the ending) | When an agent/editor requests a synopsis |
| `back-cover.md` | No-spoiler cover copy (long + short) and an Amazon-style conversion description | Jacket flap, catalog, retail listing |
| `cover-brief.md` | Art direction: palette, motifs, what to avoid, comp-cover feel | Briefing a cover designer/illustrator |
| `upstream-signals.md` | Notes confirming the pitch surfaces cleanly (premise clarity, query break point) | Self-check before submitting |

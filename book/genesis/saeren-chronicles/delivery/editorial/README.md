# Editorial / Positioning Package — The Saeren Chronicles, Book One: Hazel Academy

Submission-ready editorial and positioning materials. **Current as of build r7
(2026-06-21), ~94,000 words (93,677 exact), 18 chapters.** YA fantasy, Book One of a
planned trilogy, standalone-with-series-potential.

> **POSITIONING (read first):** This is a **quiet, interior, slow-burn, grief-driven,
> literary-leaning** YA fantasy in the loudest commercial lane (magic school +
> chosen-one). An honest professional read found the **quality is not the blocker —
> positioning is.** Every file in this package leads with the quiet voice, embodied
> grief, dual light/dark moral complexity, and found family as the **features**, and
> targets **boutique/literary agents, small presses, and a viable self-pub path** —
> NOT a loud commercial blockbuster pitch. Start with `09-positioning-honesty-note.md`.

## Comps (lead pairing)
***A Far Wilder Magic* (Allison Saft, 2022) meets *Sorcery of Thorns* (Margaret
Rogerson, 2019)** — upmarket, character-first YA fantasy. Back-pocket literary signal:
Frances Hardinge. See `05-comp-titles.md`. **Do NOT** comp Bardugo/Maas/*Fourth Wing*/
"next Harry Potter."

## Before you submit (open items)
- Fill `[AUTHOR BIO]` and `[AGENT-SPECIFIC LINE]` in `04-query-letter.md` per agent.
- Open author decisions (Alice's fate, the unsigned-note author, Raizen's affinity, the
  family-tree name) are **intentional Book-Two beats**, not query gaps — but be ready to
  speak to them as deliberate trilogy structure if asked.
- Production prep (CMYK, full front/back matter, widows/orphans pass) is in `../production/`.
